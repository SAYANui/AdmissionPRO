# AdmissionPRO 🎓
**A Centralized Digital Student Admission Management System**

AdmissionPRO is a modern, full-stack web application designed to digitize and streamline the university admission process. It simplifies the journey for students from application to enrollment while providing administrators with powerful verification tools.

## 🚀 Features
- **Centralized Dashboard:** Students can apply to multiple courses from a single portal.
- **Real-time Tracking:** Applicants can track their admission status (Pending/Approved/Rejected) instantly.
- **Admin Verification:** Robust admin panel for document verification and profile approval.
- **Digital Documentation:** Secure document upload and storage system.
- **Secure Payments:** Integrated payment confirmation (Razorpay ready).

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS ( ShadCN UI )
- **Backend:** Node.js, Express.js
- **Database & Auth:** Firebase (Firestore & Authentication)
- **Icons:** Lucide-react / React-icons
